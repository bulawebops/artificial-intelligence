## Prerequisities
Before starting this course, you should be comfortable with the following. You don't need to be an expert in any of them — a working familiarity is enough.

### Programming
Python. You can read and write basic Python — functions, classes, modules, and virtual environments (venv, pip). Most of the hands-on modules and the capstone project are written in Python.

JavaScript or TypeScript. You can read and write basic TypeScript, including async/await, modules, and typed function signatures. Several of the foundational lessons use TypeScript with the Claude Agent SDK.

Node.js and npm. You can install dependencies, run scripts, and navigate a workspace-based repo.

### Tooling
Command line. You're comfortable running commands in a terminal (macOS, Linux, or WSL on Windows).
Git. You can clone a repo, create branches, and commit changes.

A code editor such as VS Code.

### AI concepts
General experience using an LLM (Claude, ChatGPT, or similar) as a user. You don't need prior experience building with LLMs — that's what this course teaches.
Basic understanding of APIs (requests, responses, API keys)
